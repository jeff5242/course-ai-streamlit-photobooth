# AI 領航員計畫：21 天硬體自動化轉型共學實務

**—— 從意圖驅動編碼 (Vibe Coding) 到硬體系統自主掌控 ——**

-----

## 1\. 專案背景與目標

本專案針對現有「大陸製大頭貼設備」進行深度數位轉型，透過 **AI Agent (Vibe Coding)** 技術，協助長官與技術團隊在 90 天內實現系統的在地化與功能升級。

  * **技術主權化：** 拆解舊有系統「黑核」，掌握核心串接流程與硬體通訊協定。
  * **語意在地化：** 將介面與報錯訊息由簡體全面轉為符合台灣習慣之「專業繁體中文」。
  * **支付現代化：** 導入 iPad/行動支付接口，擺脫傳統紙鈔機硬體限制。
  * **開發高效化：** 建立「自然語言驅動」的開發模式，降低技術門檻，提升迭代速度。

-----

## 2\. 核心技術架構

  * **開發座艙：** VS Code + Claude 3.5 Sonnet (Agent-based IDE)
  * **互動框架：** Streamlit (用於快速原型展示與需求定義)
  * **開發模式：** **Vibe Coding** (以意圖為導向，由 AI 實現底層邏輯，人負責審核與方向)

-----


## 3\. 三個月實戰進度規劃

### 📅 第一個月：【系統解構與願景定義】

**目標：建立 AI 協作直覺，拆解現有設備邏輯，定義中文化標準。**

| 週次 | 課程主題 | 實作重點內容 | 預期產出 |
| :--- | :--- | :--- | :--- |
| **W1** | **AI 領航員啟航** | 部署 VS Code + Claude 開發環境；掌握 Vibe Coding 提問心法。 | AI 開發環境就緒 |
| **W2** | **代碼考古學** | 讀取現有大陸系統源碼，利用 AI 繪製硬體通訊流程圖（Mermaid）。 | 系統邏輯架構圖 |
| **W3** | **在地化語意重塑** | 建立「台灣繁體專業詞庫」，將 UI 介面與系統提示全面繁體化。 | 繁體化 UI 視覺規格 |
| **W4** | **需求訪談與藍圖** | 深度討論「拍攝 6 張」流程優化，定義第一個 MVP（最小可行性產品）。 | 系統改造實戰藍圖 |

### 📅 第二個月：【硬體串接與功能重構】

**目標：利用 AI 撰寫驅動邏輯，實現軟硬體（相機、印表機）的實機操控。**

| 週次 | 課程主題 | 實作重點內容 | 預期產出 |
| :--- | :--- | :--- | :--- |
| **W5** | **意圖驅動控制 (I)** | 實作相機控制：包含自動對焦、燈光補償邏輯，由 AI 優化拍攝節奏。 | 智慧相機控制模組 |
| **W6** | **意圖驅動控制 (II)** | 實作列印輸出：串接印表機 SDK，優化 6 張照片之合成排版效果。 | 自動化排版列印模組 |
| **W7** | **異常處理 Vibe 化** | 撰寫親切的繁體引導流程（如卡紙、連線中斷時的導引文字）。 | 友善引導系統 |
| **W8** | **實機整合測試** | 實機測試「啟動 -\> 支付 -\> 拍攝 -\> 列印」全流程，排除硬體衝突。 | 拍貼機 Beta 版系統 |

### 📅 第三個月：【功能進化與自主維運】

**目標：導入新式支付，建立自動化手冊，達成完全技術自主。**

| 週次 | 課程主題 | 實作重點內容 | 預期產出 |
| :--- | :--- | :--- | :--- |
| **W9** | **支付轉型計畫** | 透過 AI 串接 iPad/Web API 支付接口，實現數位化升級。 | 數位支付整合模組 |
| **W10** | **AI 加值應用實驗** | 導入 AI 自動美顏、繁體中文相框自動生成等加值功能。 | 影像增強功能模塊 |
| **W11** | **自動化文檔生成** | 讓 AI Agent 自動生成「中文化維護手冊」與「操作規範說明」。 | 設備技術維護白皮書 |
| **W12** | **成果驗收與移交** | 現場演示系統操作；總結 21 天共學成果，規劃未來擴展路徑。 | 專案正式交付 |

-----


## 4\. 預期成效 (KPI)

1.  **研發效率：** 透過 AI Agent 協作，軟體在地化與重構時間縮短 **70% 以上**。
2.  **自主能力：** 團隊無需依賴大陸廠商，具備獨立修改 UI、調整硬體邏輯之能力。
3.  **體驗升級：** 介面用語達成 100% 台灣繁體化，且流程更符合在地支付習慣。
4.  **轉型思維：** 培養長官與技術人員使用 AI 解決問題的新型態工作流。
